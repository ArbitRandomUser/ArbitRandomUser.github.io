Almost all of Javis drawing is done through the Luxor, familiarizing yourself with Luxor will go a long way in helping you use Javis. He're we'll have a small primer with Luxor.

Although Luxor can be used standalone, almost all of its functions are available when you import Javis. lets dive into some code !

using Luxor
Drawing()
move(Point(0,0))
line( Point(0,0) , Point(100,0) )
line(currentpoint(), Point(100,100) )
#check
finish()
preview()
